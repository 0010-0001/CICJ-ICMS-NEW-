🧪 How to Test Responsive Design

Step 1: Open any page (client.html, admin.html, employee.html)

Step 2: Press F12 to open Developer Tools

Step 3: Press Ctrl+Shift+M to toggle Device Toolbar

Step 4: Select devices: iPhone SE (375px), iPad (768px), Desktop (1920px)

Testing Focus: Navigation collapse, button sizes, table scrolling, form usability

📐 Design Specifications

Breakpoints:

  • Mobile Small: ≤ 320px (compact phones)
  • Mobile: ≤ 480px (standard phones)
  • Tablet: ≤ 768px (tablets, large phones)
  • Desktop Small: ≤ 1024px (small laptops)
  • Desktop: > 1024px (full desktop)
